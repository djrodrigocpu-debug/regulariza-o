# Regularização Veicular — Rodrigo Souza Filho

Landing page estática pronta para GitHub e Vercel.

## Arquivos

- `index.html`: página principal.
- `vercel.json`: configuração básica do Vercel.

## Antes de divulgar

No arquivo `index.html`, substitua:

1. `const ZAP = "5541999999999"` pelo WhatsApp real no formato `55 + DDD + número`.
2. `contato@seudominio.com.br` pelo e-mail profissional.
3. `https://SEU-DOMINIO.com.br/` pelo domínio definitivo.
4. O texto da outra área de atuação pela URL pública do segundo site, depois que ele for publicado.

Não envie o ZIP para dentro do repositório. Extraia-o e envie `index.html`, `vercel.json` e `README.md`.
